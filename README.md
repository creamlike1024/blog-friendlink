# 如何提交友链申请
[编辑 link.json](https://github.com/icedtangerine7/blog-friendlink/edit/main/link.json)，在上一个网站的 `}` 后面加上逗号 `,`，然后按类似的格式添加你的友链信息：  
```
{
  "name": "网站的名称",
  "link": "网站链接",
  "avatar": "头像链接",
  "descr": "简介"
}
```
会自动进行 folk ，编辑好了之后创建一个 Pull request 等待合并  

也可以直接在我的博客上留言，或者私信我添加友链！
